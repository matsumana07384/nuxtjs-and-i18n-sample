### 確認手順

リポジトリをクローン
```
git clone git@github.com:matsumana07384/nuxtjs-and-i18n-sample.git
```

ディレクトリに移動
```
…/nuxtjs-and-i18n-sample
```

開発モードとして起動
```
npm run dev
```
